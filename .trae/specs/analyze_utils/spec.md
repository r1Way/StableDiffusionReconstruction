# 分析 codes/utils 工具脚本规范

## 概述

为 StableDiffusionReconstruction 项目的 codes/utils 目录编写完整的工具文档。

## 变更内容

1. **分析了 37 个 Python 脚本**，理解了每个文件的功能、输入输出和参数
2. **按功能将文件分为 9 类**：数据提取、特征提取、特征处理、Ridge回归、MLP、随机森林、GNN、批量运行、可视化调试
3. **编写了详细的文档**，包含使用示例、参数说明和工作流程

## 影响范围

- 受影响的代码: `codes/utils/` 目录下所有 Python 脚本
- 生成的文档: `docs/codes_utils_readme.md`

## 需求说明

### 需求：完整的工具文档

系统应提供详细的工具脚本文档，包含：

- 每个脚本的功能描述
- 使用示例
- 输入输出说明
- 参数列表
- 版本演进说明

#### 场景：用户使用工具

- **当** 用户想要了解某个工具脚本的用途
- **则** 用户可以查看文档获取详细信息

#### 场景：用户选择合适的模型

- **当** 用户想要选择合适的脑解码模型
- **则** 文档应提供模型对比和适用场景说明
